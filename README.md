--------------성수 쌤을 위한 가장 빠른 테스트 방법--------------
1. 게임을 실행한 뒤 대기화면이 나오면 Tab을 누르세요.
2. Player 1의 공격 순서입니다. 커맨드를 입력할 수 있습니다. 가장 간단한 스킬인 태스트 스킬을 시전하기 위해 Space Bar를 누른 뒤, Tab 키를 누르세요.
3. 이제 Player 2의 방어 순서입니다. Tab 키를 눌러 커맨드 입력 화면에 진입하세요.
4. 방어는 공격 커맨드를 역순으로 하면 됩니다. Player 1의 공격 커맨드가 Space Bar를 누르는 것 하나였으므로, Player 2는 Space Bar를 누르는 것만으로 방어할 수 있습니다. Space Bar를 누른 후 Tab 키를 눌러 방어하세요.
5. 공격 데미지와 방어력은 입력하고 남은 시간(10초 - 입력 시간)에 비례합니다. 이를 반영하여 애니메이션과 함께 데미지(최종 공격 데미지 - 방어력)를 정산합니다.
6. Player 1의 공격이 끝났으므로 Player 2의 공격 차례입니다. 위 과정을 반복하면 됩니다.
7. MP(노란색 바)가 가득 차면(100) 궁극기를 사용할 수 있습니다. 궁극기는 무조건 최대 데미지로 반영되며 방어할 수 없습니다. 궁극기 사용 방법은 아래를 참고하세요.
----------------------------------------------------------------


One By One 상세 설명
1. 게임 화면 및 기본 목표: 화면 양쪽에는 플레이어 캐릭터가 서 있고, 화면 하단에는 가상의 손이 있습니다.
목표: 상대방의 HP(체력)를 0으로 만들면 승리합니다.
방식: 공격과 방어 모두 '제한 시간(10초)' 안에 정확한 손동작을 입력해야 하며, 빨리 입력할수록 데미지가 강력해지거나 방어력이 올라갑니다.

2. 조작법 (내 손가락 움직이기)
이 게임의 핵심은 키보드 숫자키로 화면 속 손가락을 접었다 펴는 것입니다.

왼손 조작 (키보드 왼쪽 숫자키): 1, 2, 3, 4
1: 검지 접기
2: 중지 접기
3: 약지 접기
4: 소지(새끼) 접기

오른손 조작 (키보드 오른쪽 숫자키): 7, 8, 9, 0
7: 검지 접기
8: 중지 접기
9: 약지 접기
0: 소지(새끼) 접기

동작 확정: Space Bar (현재 손 모양을 하나의 동작으로 입력)턴 종료 / 스킬 발동: Tab (입력한 동작들을 모아 스킬 발동)

※중요: 키를 누르고 있으면 손가락이 접히고, 떼면 펴집니다. 원하는 손 모양을 만든 상태(키를 꾹 누른 상태)에서 Space Bar를 쳐야 입력됩니다.

3. 게임 진행 단계
게임은 [대기] -> [공격 턴] -> [방어 턴] -> [데미지 정산] 순서로 진행됩니다.
1단계: 대기 및 시작
게임이 시작되면 대기 화면이 나옵니다. Tab 키를 눌러 게임을 시작하세요.

2단계: 공격하기 (나의 턴)
공격 차례가 되면 10초의 카운트다운이 시작됩니다. 원하는 스킬 레시피대로 손동작을 입력해야 합니다.
예시: '손 풀기' 스킬 쓰기 (레시피: 왼손 주먹 → 오른손 주먹)
[왼손 주먹] 만들기: 1, 2, 3, 4 키를 동시에 꾹 누릅니다. (왼손가락 다 접힘)
입력: 키를 누른 상태에서 Space Bar를 탁! 칩니다. (화면에 '왼손 주먹'이라고 뜹니다.)
손 떼기: 누르던 키를 뗍니다.
[오른손 주먹] 만들기: 7, 8, 9, 0 키를 동시에 꾹 누릅니다.
입력: 키를 누른 상태에서 Space Bar를 탁! 칩니다.
발동: 모든 동작이 끝났으면 Tab 키를 누릅니다. 제한 시간(10초)이 줄어들수록 데미지가 약해집니다. 최대한 빠르게 입력해야 상대에게 큰 피해를 줄 수 있습니다.

3단계: 방어하기 (상대의 턴)
상대방이 공격을 성공하면, 당신에게 방어 기회가 주어집니다.
방어 규칙: 상대방이 사용한 스킬의 동작을 거꾸로(역순으로) 입력해야 합니다.
예를 들어 상대가 [가위] → [바위] 순서로 공격했다면, 당신은 [바위] → [가위] 순서로 입력하고 Tab을 눌러야 막아집니다. 방어 역시 빨리 입력할수록 방어력이 높아져 데미지를 덜 입습니다.

4단계: 필살기 (MP 100)
공격을 하거나 맞다 보면 노란색 MP 게이지가 찹니다. MP가 100이 되면 일반 공격 대신 궁극기를 쓸 수 있습니다.궁극기는 데미지가 매우 강력하며, 상대방이 방어할 수 없습니다.

4. 스킬 레시피 (이걸 외워야 합니다!)
게임 코드에 등록된 동작 만드는 법입니다. (키를 누르면 접힘, 안 누르면 펴짐)

보자기[아무것도 안 누름]: 열 손가락 다 펴기
왼손 주먹[1, 2, 3, 4]: 왼손 꽉 쥐기
오른손 주먹[7, 8, 9, 0]: 오른손 꽉 쥐기
바위[1~4 + 7~0 (전부 다)]: 양주먹 꽉 쥐기
가위[1, 2 + 9, 0]: 왼손 검지중지 접고, 오른손 약지소지 접기
피스 사인[2, 3 + 8, 9]: (조금 복잡함) 각 손의 중지/약지 조작
왼손 가리키기[1, 2, 3 + 8, 9, 0 + 7]: 왼손 소지만 펴고 오른손은 검지 펴기 등
HYPER TEXT[2, 4 + 7, 9]: 각 손의 중지/소지 펴기(궁극기 전용 동작)

5. 스킬 콤보 리스트 (공격 기술표)
아래 순서대로 동작을 입력(Space)하고 마지막에 발동(Tab)하세요.
손 풀기: 왼손 주먹 → 오른손 주먹
가위바위보: 가위 → 바위 → 보자기
삿대질: 왼손 가리키기 → 오른손 가리키기 → 가위
술식전개: 가위 → 보자기 → 가위 → 피스 사인 → 바위
테스트: 보자기(Space만 누르면 됩니다.)
궁극기 (MP 100일 때만 가능): HYPER TEXT (동작 1개)

6. 게임 진행 요약
Tab을 눌러 턴을 시작한다.
'손 풀기'나 '가위바위보' 같은 스킬 콤보를 최대한 빠르게 입력한다.
상대가 공격해오면, 상대가 낸 동작을 기억했다가 거꾸로 입력해서 막는다.
MP가 다 차면 궁극기를 사용한다.
